# Subtitles Tools 插件前端设计系统

这个目录只服务 `webui` 内部，不追求跨仓库复用。

## 目标

- 统一设置页和注入字幕管理页的视觉 token、组件命名和状态表达。
- 把高频样式决策收口到共享层，避免 Tailwind utility 在业务组件里无序扩散。
- 让后续新增模块优先复用共享组件，而不是回到字符串模板或一次性样式。

## 组成

- `theme.css`：Tailwind v4 入口、主题 token、根节点作用域与通用基础样式。
- `components.tsx`：基础 UI 组件，包括按钮、徽标、卡片、状态条、字段容器、空状态等。

## 使用约定

- 所有颜色、圆角、阴影、字号、状态色优先使用 token，不直接写匿名任意值。
- 业务页面可以组合 utility class，但公共视觉模式应先沉淀到共享组件。
- 所有类名前缀统一为 `st-`，避免与 Jellyfin 全局样式冲突。
